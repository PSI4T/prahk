NEW BETA VERSION v0.2.8.9:
---------------------------------------------------------------------------------------------
1.Добавлена возможность ставить 2 режима экрана (Окно/Полноэкранный).
1.1 Чтобы окно возвращало в полноэкранный режим. Поставьте галочку возле WM.
1.2 Для того чтобы у вас корректно работало все. Разрешение игры должно совпадать с виндой.
1.3 Перед запуском игры, нажмите один раз Alt+F2
2.Исправлена отыгровка с чулками. Теперь клист 0 ставится сам.
3.Добавлены новые команды для отыгрывания оружия. И использование наркотиков.
3.1 Зажмите одну и цифрф от 1 до 4. И покрутите колесо мыши -
вверх (Если хотите достать), вниз (Если хотите убрать).
3.2 [1] - Пистолет, [2] - Автомат, [3] - Винтовка, [4] - Холодное, [5] - Наркотики.
3.3 Наркотики + Вверх - Выбор (1-5грамм), Наркотики + Низ - 6 грамм.
4.Улучшено принятие наркотиков в режиме выбора. Теперь не нужно нажимать Еnter.
