NEW BETA VERSION v0.2.9.5:
-----------------------------------------------------------------------------------------------------------------
1.Исправлена проблема с часами. Дом иногда не засчитывало.

9.Изменены команды в "Общее/Настройки".

P.S Для переключения фракции. Включите NumLock. Зажмите [NDiv(/) + NDot(.)] и выберите пункт.
Для того чтобы посмотреть в помощи команды для фракции/семьи. Меняйте NumLock (вкл/откл).
Эту версию я исправил. И выпустил для тестирования нового функционала.
Если будут какие-то проблемы с неудобством. Пишите в поддержку. [NDiv(/) + F2]
